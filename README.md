# 👨‍💻 Oscar Murguías — CV Personal

Sitio web estático de CV personal construido con **React + Vite**.  
Incluye animaciones, diseño responsivo y secciones completas de perfil profesional DevOps.

---

## 🚀 Levantar el proyecto en local

### Requisitos previos

Asegúrate de tener instalados:

- [Node.js](https://nodejs.org/) **v18 o superior**
- **npm** v9+ (viene incluido con Node.js)

Verifica tu versión con:

```bash
node -v
npm -v
```

---

### Pasos de instalación

1. **Clona el repositorio**

```bash
git clone <URL_DEL_REPOSITORIO>
cd awsclass
```

2. **Instala las dependencias**

```bash
npm install
```

3. **Inicia el servidor de desarrollo**

```bash
npm run dev
```

4. **Abre el navegador**

Visita [http://localhost:5173](http://localhost:5173)

---

## 🏗️ Build para producción

```bash
npm run build
```

Los archivos optimizados se generan en la carpeta `dist/`.

Para previsualizar el build de producción localmente:

```bash
npm run preview
```
